# Proposal Kerja Sama Koperasi & SPPG MBG Lestari

Repositori ini memuat aplikasi Flutter dan dokumen rancangan proposal kerja sama antara **Koperasi Kelurahan Air Dingin, Kecamatan Bukit Raya, Kota Pekanbaru** dengan **SPPG MBG Lestari Kelurahan Air Dingin**.

## Dokumen Proposal
Silakan lihat dokumen lengkapnya pada file: [`PROPOSAL_KERJA_SAMA.md`](./PROPOSAL_KERJA_SAMA.md)

Proposal ini dirancang untuk mensinergikan pemberdayaan ekonomi masyarakat (koperasi) dengan penyediaan rantai pasok dan tenaga kerja lokal untuk kelancaran operasional program gizi.

## Tentang Aplikasi (Flutter)
Repositori ini juga disiapkan sebagai wadah aplikasi lintas platform (Android, iOS, Web, Desktop) yang dapat digunakan nantinya sebagai sistem digitalisasi untuk:
- Manajemen Rantai Pasok (Supply Chain) Koperasi.
- Pengelolaan pesanan (Purchase Order) antara SPPG MBG Lestari dan Koperasi.
- Pelaporan dan transparansi bagi anggota.

### Persiapan Sistem
Untuk menjalankan aplikasi scaffold awal:
```bash
flutter pub get
flutter run
```
